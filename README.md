#### Features
- typescript used on both sides
    - client
    - server
- used libraries
    - behavior subject
    - reactive forms

#### Prepare to work
- `cd client` 
- `npm install` 
- `cd ../server`
- `npm install typescript --global`
- `npm install `
- `cd ..`
 

#### How to start
- `cd client`
- `ng build`
- `cd ../server`
- `npm run serve`
